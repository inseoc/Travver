# Travver
Travel + Trigger 서비스로, 오사카 여행 충동을 일으키기 위해 만들어진 앱 서비스

# 1. 프로젝트 개요
## 1.1 프로젝트명
Travver ver.오사카
## 1.2 프로젝트 목적
- AI 에이전트를 활용한 개인 맞춤형 오사카 여행 컨설팅 및 예산 관리 서비스 개발
- 사용자와의 대화형 인터페이스를 통해 자연스러운 여행 계획 수립 지원
- 효율적인 예산 관리를 통한 최적의 여행 경험 제공
## 1.3 타겟 사용자
- 오사카 여행을 계획 중인 개인 여행자
- 여행 예산을 효율적으로 관리하고자 하는 사용자
- 현지 정보에 대한 전문적인 조언이 필요한 사용자

# 2. 핵심 기능 상세
## 2.1 AI 여행 컨설턴트 기능
### 2.1.1 기능 개요
- 사용자와 대화형 인터페이스를 통해 상호작용하는 AI 에이전트
- 오사카 여행에 특화된 맞춤형 조언 및 추천 제공
- 사용자의 선호도와 제약 조건을 고려한 개인화된 여행 계획 수립
### 2.1.2 세부 기능
1. 사용자 프로필링
2. 초기 대화를 통한 여행 스타일, 선호도, 관심사 파악
3. 여행 기간, 동반자 유형, 이동 수단 등 기본 정보 수집
4. 사용자의 과거 여행 경험 분석
5. 맞춤형 질의응답
6. 사용자의 응답에 따른 적응형 질문 생성
7. 오사카 여행 관련 구체적인 정보 요청 및 제공
8. 사용자의 불명확한 요구사항 명확화
9. 오사카 특화 추천
10. 사용자 선호도에 맞는 오사카 명소 추천
11. 계절, 날씨, 현지 이벤트 등을 고려한 방문 일정 제안
12. 현지 음식, 쇼핑, 문화 체험 등 테마별 추천
13. 일정 계획 지원
14. 효율적인 동선을 고려한 일별 방문 계획 수립
15. 관광지 혼잡도, 운영 시간 등을 고려한 최적 방문 시간 제안
16. 예상 소요 시간 및 이동 방법 안내
### 2.1.3 AI Agent 역할
- 컨텍스트 인식: 대화 맥락을 이해하고 일관된 대화 흐름 유지
- 개인화 엔진: 사용자 정보를 바탕으로 개인화된 추천 제공
- 지식 베이스: 오사카 관련 정보(관광지, 음식, 교통 등) 제공
- 자연어 처리: 사용자의 질문 의도 파악 및 자연스러운 응답 생성
- 학습 메커니즘: 사용자 피드백을 통한 추천 알고리즘 개선
## 2.2 예산 관리 어드바이저 기능
### 2.2.1 기능 개요
- 사용자의 오사카 여행 예산을 효율적으로 관리하는 AI 기반 도구
- 항목별 예산 배분 및 지출 최적화 제안
- 예산 제약 내에서 최대 만족도를 얻을 수 있는 여행 계획 수립
### 2.2.2 세부 기능
1. 예산 설정 및 분석
2. 총 예산 설정 및 항목별(숙박, 식사, 교통, 관광, 쇼핑 등) 예산 배분
3. 오사카 현지 물가 정보를 반영한 예상 지출 분석
4. 예산 적정성 평가 및 조정 제안
5. 지출 최적화 제안
6. 사용자 선호도와 예산을 고려한 최적의 숙박 옵션 추천
7. 비용 효율적인 교통 수단 및 패스 추천
8. 가성비 좋은 식당 및 쇼핑 장소 추천
9. 예산 시뮬레이션
10. 계획된 일정에 따른 예상 지출 시뮬레이션
11. 예산 초과 시 대안 활동 및 절약 방안 제시
12. 항목별 지출 균형 조정 제안
13. 환율 및 결제 조언
14. 현재 엔화 환율 정보 제공
15. 오사카에서의 효율적인 결제 방법 안내
16. 팁 문화, 세금 환급 등 현지 금융 정보 제공
### 2.2.3 AI Agent 역할
- 예산 분석기: 사용자 예산과 선호도를 분석하여 최적의 지출 방안 도출
- 가격 예측기: 오사카 현지 가격 데이터를 기반으로 정확한 지출 예측
- 우선순위 엔진: 사용자의 선호도에 따른 지출 우선순위 설정
- 대안 생성기: 예산 제약 시 유사한 경험을 제공할 수 있는 대안 제시
- 학습 시스템: 사용자 피드백을 통한 예산 추천 알고리즘 개선
# 3. 오사카 특화 데이터 및 API
## 3.1 오사카 관광 데이터
- 관광 명소 데이터
- 주요 관광지 정보(위치, 운영 시간, 입장료, 특징 등)
- 계절별 추천 명소 및 이벤트 정보
- 현지인 추천 숨은 명소 정보
- 음식 및 쇼핑 데이터
- 오사카 대표 음식점 정보(위치, 가격대, 메뉴 등)
- 쇼핑 장소 정보(상점가, 백화점, 아울렛 등)
- 음식 및 쇼핑 관련 현지 팁
- 교통 데이터
- 대중교통 노선도 및 운행 정보
- 교통 패스 종류 및 가격 정보
- 주요 지점 간 이동 시간 및 방법
## 3.2 추천 API 및 데이터 소스
- 지도 API
  - Google Maps API: 위치 정보, 경로 안내, 거리 계산
  - MapBox API: 커스텀 지도 시각화, 오프라인 지도 지원
  - OpenStreetMap: 오픈소스 지도 데이터
- 날씨 API
  - OpenWeatherMap API: 오사카 지역 일기 예보 및 현재 날씨 정보
  - Weather API (weatherapi.com): 상세 날씨 정보 및 예보
  - Japan Meteorological Agency API: 일본 기상청 공식 데이터
- 여행 정보 API 및 데이터 소스
  - Osaka Tourism Official Website: 공식 관광 정보 및 이벤트 데이터
  - Jalan API: 일본 숙박 및 관광 정보
  - Rakuten Travel API: 숙박 시설 정보 및 가격
  - Tabelog API: 일본 식당 정보 및 리뷰
- 환율 및 금융 API
  - Exchange Rate API: 실시간 환율 정보
  - Currency Converter API: 통화 변환 기능
# 4. 기술 스택
## 4.1 백엔드
- 언어: Python 3.9+
- 프레임워크: FastAPI
- 로컬 서버: Uvicorn
- 데이터베이스: SQLite (로컬 저장용)
- AI/ML 라이브러리:
  - OpenAI API (GPT 모델 활용)
  - Hugging Face Transformers (로컬 NLP 모델)
  - scikit-learn (데이터 분석 및 추천 알고리즘)
- NLTK (자연어 처리)
## 4.2 프론트엔드
- 프레임워크: React Native
- 상태 관리: Redux 또는 Context API
- UI 컴포넌트: React Native Paper
- 지도 컴포넌트: React Native Maps
- 차트/시각화: Victory Native 또는 React Native SVG Charts
## 4.3 데이터 관리
- 로컬 저장소: AsyncStorage, SQLite
- 데이터 형식: JSON
- 오프라인 데이터 캐싱: Redux Persist

# 8. 결론
Travver ver.오사카는 AI 여행 컨설턴트와 예산 관리 어드바이저 기능을 핵심으로 하는 개인화된 여행 계획 앱입니다. 사용자와의 자연스러운 대화를 통해 오사카 여행에 대한 전문적인 조언을 제공하고, 효율적인 예산 관리를 통해 최적의 여행 경험을 지원합니다.
이 토이 프로젝트는 파이썬과 React Native를 활용하여 로컬 환경에서 개발되며, 클라우드나 외부 DB에 의존하지 않고도 AI 에이전트의 핵심 기능을 구현합니다. 오사카라는 특정 지역에 집중함으로써 깊이 있는 정보 제공과 정확한 추천이 가능하며, 향후 다른 도시로의 확장 가능성도 열려 있습니다.
12주간의 개발 일정을 통해 기본 설계부터 최종 완료까지 체계적으로 진행하며, 각 단계별로 핵심 기능을 우선적으로 구현하여 실용적인 앱을 완성할 계획입니다.
