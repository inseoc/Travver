# Travver
Travel + Trigger 서비스로, 오사카 여행 충동을 일으키기 위해 만들어진 앱 서비스

# 1. 프로젝트 개요
## 1.1 프로젝트명
Travver ver.오사카
## 1.2 프로젝트 목적
- AI 에이전트를 활용한 개인 맞춤형 오사카 여행 컨설팅 서비스 개발
- 사용자와의 대화형 인터페이스를 통해 자연스러운 여행 계획 수립 지원

## 1.3 타겟 사용자
- 오사카 여행을 계획 중인 개인 여행자
- 현지 정보에 대한 전문적인 조언이 필요한 사용자

# 2. 핵심 기능 상세
## 2.1 AI 여행 컨설턴트 기능
### 2.1.1 기능 개요
- 사용자와 대화형 인터페이스를 통해 상호작용하는 AI 에이전트
- 오사카 여행에 특화된 맞춤형 조언 및 추천 제공
- 사용자의 선호도와 제약 조건을 고려한 개인화된 여행 계획 수립
### 2.1.2 세부 기능
1. 초기 대화를 통한 여행 스타일, 선호도, 관심사 파악
2. 여행 기간, 동반자 유형, 이동 수단 등 기본 정보 수집
3. 오사카 여행 관련 구체적인 정보 요청 및 제공
4. 사용자의 불명확한 요구사항 명확화
5. 오사카 특화 추천
6. 사용자 선호도에 맞는 오사카 명소 추천
7. 계절, 날씨, 현지 이벤트 등을 고려한 방문 일정 제안
8. 현지 음식, 쇼핑, 문화 체험 등 테마별 추천
9. 일정 계획 지원
10. 효율적인 동선을 고려한 일별 방문 계획 수립
11. 관광지 혼잡도, 운영 시간 등을 고려한 최적 방문 시간 제안
12. 예상 소요 시간 및 이동 방법 안내

# 3. 오사카 특화 데이터 및 API
## 3.1 오사카 관광 데이터
- 관광 명소 데이터
- 주요 관광지 정보(위치, 운영 시간, 입장료, 특징 등)
- 계절별 추천 명소 및 이벤트 정보
- 현지인 추천 숨은 명소 정보
- 음식 및 쇼핑 데이터
- 오사카 대표 음식점 정보(위치, 가격대, 메뉴 등)
- 쇼핑 장소 정보(상점가, 백화점, 아울렛 등)
- 음식 및 쇼핑 관련 현지 팁
- 교통 데이터
- 대중교통 노선도 및 운행 정보
- 주요 지점 간 이동 시간 및 방법
## 3.2 추천 API 및 데이터 소스
- 지도 API
  - Google Maps API: 위치 정보, 경로 안내, 거리 계산
# 4. 기술 스택
## 4.1 백엔드
- 언어: Python 3.9+
- 프레임워크: FastAPI
- 로컬 서버: Uvicorn
- AI/ML 라이브러리:
  - OpenAI API (GPT 모델 활용)
## 4.2 프론트엔드
- 프레임워크: Flutter
- UI 컴포넌트: Flutter

- 3-6주차: 핵심 기능 개발

# 8. 결론
Travver ver.오사카는 AI 여행 컨설턴트 기능을 핵심으로 하는 개인화된 여행 계획 앱입니다. 사용자와의 자연스러운 대화를 통해 오사카 여행에 대한 전문적인 조언을 제공하여 최적의 여행 경험을 지원합니다.
이 토이 프로젝트는 파이썬과 Flutter를 활용하여 로컬 환경에서 개발되며, 클라우드나 외부 DB에 의존하지 않고도 AI 에이전트의 핵심 기능을 구현합니다. 오사카라는 특정 지역에 집중함으로써 깊이 있는 정보 제공과 정확한 추천이 가능하며, 향후 다른 도시로의 확장 가능성도 열려 있습니다.
