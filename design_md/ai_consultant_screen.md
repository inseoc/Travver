# AI 컨설턴트 화면 디자인 명세

## 화면 기본 정보
- **프레임 사이즈**: 390x844 (iPhone 13 Pro)
- **배경색**: 밝은 회색(#F9FAFB)
- **상태 바**: 다크 모드

## 구성 요소
1. **상단 앱 바**
   - 높이: 56px
   - 배경색: 진한 남색(#1E3A8A)
   - 요소:
     - 뒤로 가기 아이콘(좌측): 24x24px, 흰색(#FFFFFF)
     - 제목(중앙): "AI 여행 컨설턴트", Noto Sans KR Medium, 18px, 흰색(#FFFFFF)
     - 정보 아이콘(우측): 24x24px, 흰색(#FFFFFF)

2. **AI 캐릭터 프로필**
   - 높이: 80px
   - 배경색: 흰색(#FFFFFF)
   - 그림자 효과: 약한 드롭 쉐도우
   - 요소:
     - AI 캐릭터 아바타(좌측): 48x48px, 원형
     - 이름: "트래버(Travver)", Noto Sans KR Medium, 16px, #1F2937
     - 상태: "오사카 여행 전문가", Noto Sans KR Regular, 14px, #4B5563
     - 온라인 상태 표시: 작은 초록색 점

3. **대화 영역**
   - 배경: 패턴 있는 연한 배경(#F9FAFB)
   - 패딩: 16px
   - 메시지 버블:
     - AI 메시지:
       - 배경색: 흰색(#FFFFFF)
       - 텍스트 색: #1F2937
       - 정렬: 좌측
       - 코너 라운딩: 16px (우측 상단만 뾰족)
     - 사용자 메시지:
       - 배경색: 진한 남색(#1E3A8A)
       - 텍스트 색: 흰색(#FFFFFF)
       - 정렬: 우측
       - 코너 라운딩: 16px (좌측 상단만 뾰족)
     - 시간 표시: 각 메시지 아래, #9CA3AF, 12px

4. **추천 질문 영역**
   - 가로 스크롤 리스트
   - 위치: 입력 필드 위
   - 버튼 스타일:
     - 배경: 흰색(#FFFFFF)
     - 테두리: #E5E7EB
     - 텍스트: "오사카 필수 관광지는?", #1F2937
     - 크기: 자동 조절 (내용에 맞게)
     - 패딩: 12px
     - 코너 라운딩: 20px

5. **입력 영역**
   - 높이: 64px
   - 배경색: 흰색(#FFFFFF)
   - 그림자 효과: 위쪽으로 향하는 드롭 쉐도우
   - 요소:
     - 입력 필드:
       - 배경: #F3F4F6
       - 텍스트: #1F2937
       - 플레이스홀더: "질문을 입력하세요...", #9CA3AF
       - 코너 라운딩: 24px
     - 전송 버튼:
       - 아이콘: 종이비행기, 진한 남색(#1E3A8A)
       - 크기: 48x48px
       - 배경: 원형, 흐린 파란색(#EFF6FF)

6. **여행 정보 카드 (AI 응답에 포함)**
   - 크기: 300x180px
   - 배경: 흰색(#FFFFFF)
   - 테두리: #E5E7EB
   - 코너 라운딩: 12px
   - 요소:
     - 이미지: 상단 부분, 관광지 이미지
     - 제목: 하단 부분, "오사카성", Noto Sans KR Bold, 16px, #1F2937
     - 설명: 간략한 설명, Noto Sans KR Regular, 14px, #4B5563
     - 버튼: "상세 정보", 작은 텍스트 버튼, #1E3A8A

## 상호작용
- **뒤로 가기 아이콘**: 탭하면 이전 화면으로 이동
- **정보 아이콘**: 탭하면 AI 컨설턴트 정보 및 사용 팁 팝업
- **추천 질문 버튼**: 탭하면 해당 질문이 자동으로 입력 필드에 입력
- **전송 버튼**: 탭하면 입력한 질문 전송
- **여행 정보 카드**: 탭하면 해당 장소/정보 상세 페이지로 이동
- **스크롤**: 대화 내용이 많아지면 자동 스크롤, 위로 스와이프하여 이전 대화 확인 가능
