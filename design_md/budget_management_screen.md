## 상호작용
- **뒤로 가기 아이콘**: 탭하면 이전 화면으로 이동
- **설정 아이콘**: 탭하면 예산 설정 옵션 팝업
- **편집 버튼(총 예산)**: 탭하면 총 예산 수정 모달 표시
- **항목 편집 아이콘**: 탭하면 해당 항목 예산 수정 모달 표시
- **예산 추천 버튼**: 탭하면 AI 예산 추천 프로세스 시작
- **항목별 예산 차트**: 각 섹션 탭 시 해당 항목 상세 정보 표시
- **스크롤**: 콘텐츠가 화면보다 길 경우 세로 스크롤 지원
