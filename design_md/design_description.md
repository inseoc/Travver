# Travver 앱 디자인 설명

## 디자인 컨셉
- **모던 & 미니멀**: 깔끔하고 직관적인 UI로 사용자 경험을 극대화
- **일본 & 오사카 감성**: 오사카의 특징적인 요소와 일본 디자인 미학 반영
- **사용자 친화적**: 복잡한 여행 계획과 예산 관리를 쉽고 재미있게 

## 컬러 팔레트
- **주요 색상(Primary)**: #1E3A8A - 오사카의 밤하늘을 연상시키는 진한 남색
- **보조 색상(Secondary)**: #F97316 - 도톤보리의 활기찬 분위기를 표현하는 따뜻한 주황색
- **강조 색상(Accent)**: #DC2626 - 일본의 상징적인 빨간색
- **배경 색상(Background)**: #FFFBEB - 따뜻하고 친근한 느낌의 크림색
- **텍스트 색상(Text)**: 짙은 회색(#1F2937)과 흰색(#FFFFFF)

## 타이포그래피
- **헤더**: Noto Sans KR Bold, 24-28px
- **서브헤더**: Noto Sans KR Medium, 18-20px
- **본문**: Noto Sans KR Regular, 14-16px
- **작은 텍스트**: Noto Sans KR Light, 12px

## 아이콘 및 이미지
- 선형(Line) 아이콘과 필 처리된(Filled) 아이콘 조합 사용
- 오사카 랜드마크를 단순화한 일러스트레이션
- 실제 오사카 이미지를 블러 처리하거나 오버레이하여 배경으로 활용

## 화면 목록
1. **스플래시 화면**
   - 앱 로고와 브랜드명
   - 간단한 애니메이션 효과
   - 오사카를 상징하는 요소

2. **온보딩 화면**
   - 앱의 핵심 기능 소개 (AI 컨설턴트, 예산 관리)
   - 사용 방법 간략 설명
   - 시작하기 버튼

3. **홈 화면**
   - 사용자 인사말 및 프로필
   - AI 컨설턴트 대화 바로가기
   - 예산 관리 도구 바로가기
   - 추천 오사카 여행 정보 카드

4. **AI 컨설턴트 화면**
   - 챗봇 인터페이스
   - 대화 입력 필드
   - 추천 질문 버튼
   - 이전 대화 내역

5. **예산 관리 화면**
   - 총 예산 표시
   - 항목별 예산 할당 차트
   - 지출 추적 도구
   - 예산 조정 기능

6. **여행 계획 화면**
   - 일정표 뷰
   - 지도 통합 뷰
   - 장소별 상세 정보
   - 계획 공유 기능
