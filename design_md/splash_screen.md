# 스플래시 화면 디자인 명세

## 화면 기본 정보
- **프레임 사이즈**: 390x844 (iPhone 13 Pro)
- **배경색**: 그라데이션 (#1E3A8A → #0F172A)
- **애니메이션**: 페이드 인 효과 (2초)

## 구성 요소
1. **로고**
   - 중앙 배치
   - 크기: 200x200px
   - 디자인: 여행 가방과 비행기 모티프를 활용한 심플한 로고
   - 색상: 흰색(#FFFFFF) + 주황색(#F97316) 포인트

2. **앱 이름**
   - 로고 아래 배치
   - 폰트: Noto Sans KR Bold, 36px
   - 텍스트: "Travver"
   - 색상: 흰색(#FFFFFF)

3. **태그라인**
   - 앱 이름 아래 배치
   - 폰트: Noto Sans KR Light, 16px
   - 텍스트: "오사카 여행의 모든 것"
   - 색상: 연한 회색(#E5E7EB)

4. **오사카 랜드마크 일러스트**
   - 하단에 실루엣 형태로 배치
   - 오사카성, 도톤보리 글리코 간판, 츠텐카쿠 등 오사카 상징물
   - 반투명 처리 (opacity: 30%)
   - 색상: 흰색(#FFFFFF)

5. **로딩 인디케이터**
   - 하단 중앙에 배치
   - 스타일: 점선 원형 프로그레스
   - 색상: 주황색(#F97316)
   - 애니메이션: 회전 효과

## 상호작용
- 스플래시 화면은 자동으로 3초 후 온보딩 또는 홈 화면으로 전환
- 화면 전환 시 페이드 아웃 효과 적용
