- **프레임 사이즈**: 390x844 (iPhone 13 Pro)
- **배경색**: 크림색(#FFFBEB)
- **상태 바**: 라이트 모드

   - 배경색: 흰색(#FFFFFF)

   - 요소:
     - 타이틀(중앙): "여행 계획", Noto Sans KR Medium, 18px, #1F2937

     - 편집 아이콘: 연필 아이콘, #9CA3AF

3. **일정 선택 탭**
   - 높이: 48px
   - 배경색: 흰색(#FFFFFF)
   - 패딩: 8px
   - 스크롤 가능한 가로 탭

     - 선택된 탭:
       - 배경색: #1E3A8A
       - 텍스트 색: 흰색(#FFFFFF)

     - 선택되지 않은 탭:
       - 배경색: #F3F4F6
       - 텍스트 색: #4B5563
       - "2일차", Noto Sans KR Medium, 14px
     - 코너 라운딩: 20px
     - 패딩: 12px 16px

4. **지도 뷰**

     - 경로선: 장소 간 이동 경로를 표시하는 선
     - 확대/축소 버튼: 우측 하단에 배치
     - 전체 화면 버튼: 좌측 하단에 배치

5. **일정 리스트**

       - 장소 이름: "오사카성", Noto Sans KR Bold, 16px, #1F2937

       - 추가 정보 아이콘: 화살표 또는 더보기 아이콘, #9CA3AF

   - 스타일: FAB(Floating Action Button)
   - 배경색: 진한 남색(#1E3A8A)
   - 아이콘: 플러스(+), 흰색(#FFFFFF)
   - 크기: 56x56px
   - 그림자 효과: 중간 강도 드롭 쉐도우

## 상호작용
- **일정 공유 아이콘**: 탭하면 일정 공유 옵션 표시
- **여행 기간 편집 아이콘**: 탭하면 여행 기간 수정 모달 표시
- **일정 선택 탭**: 탭하면 해당 일차의 일정으로 전환
- **지도 뷰**: 
  - 마커 탭 시 해당 장소 정보 간략히 표시
  - 확대/축소 버튼으로 지도 확대/축소
  - 전체 화면 버튼으로 지도 전체 화면 보기
- **일정 리스트 아이템**: 탭하면 해당 장소 상세 정보 페이지로 이동
- **추가 버튼**: 탭하면 새로운 일정 추가 모달 표시
- **스크롤**: 일정 리스트가 화면보다 길 경우 세로 스크롤 지원
