# 홈 화면 디자인 명세

## 화면 기본 정보
- **프레임 사이즈**: 390x844 (iPhone 13 Pro)
- **배경색**: 크림색(#FFFBEB)
- **상태 바**: 다크 모드

## 구성 요소
1. **상단 앱 바**
   - 높이: 56px
   - 배경색: 흰색(#FFFFFF)
   - 그림자 효과: 약한 드롭 쉐도우
   - 요소:
     - 앱 로고(좌측): 24x24px
     - 앱 이름(중앙): "Travver", Noto Sans KR Medium, 18px, #1E3A8A
     - 사용자 프로필 아이콘(우측): 32x32px, 원형

2. **사용자 인사말 섹션**
   - 패딩: 24px
   - 텍스트 1: "안녕하세요, [사용자 이름]님", Noto Sans KR Bold, 24px, #1F2937
   - 텍스트 2: "오사카 여행을 함께 계획해볼까요?", Noto Sans KR Regular, 16px, #4B5563
   - 배경: 부드러운 물결 패턴 (오사카 컬러)

3. **AI 컨설턴트 카드**
   - 크기: 342x120px
   - 코너 라운딩: 16px
   - 배경색: 진한 남색(#1E3A8A)
   - 패딩: 20px
   - 요소:
     - 아이콘: AI/로봇 아이콘, 48x48px
     - 제목: "AI 여행 컨설턴트", Noto Sans KR Bold, 20px, 흰색(#FFFFFF)
     - 설명: "질문하고 맞춤형 여행 계획을 받아보세요", Noto Sans KR Regular, 14px, #E5E7EB
     - 버튼: "대화하기", 흰색 테두리, 투명 배경, 우측 하단 배치

4. **예산 관리**
   - 크기: 342x120px
   - 코너 라운딩: 16px
   - 배경색: 따뜻한 주황색(#F97316)
   - 패딩: 20px
   - 요소:
     - 아이콘: 지갑/차트 아이콘, 48x48px
     - 제목: "예산 관리 어드바이저", Noto Sans KR Bold, 20px, 흰색(#FFFFFF)
     - 설명: "효율적인 여행 예산 관리를 도와드립니다", Noto Sans KR Regular, 14px, #FFFBEB
     - 버튼: "예산 설정하기", 흰색 테두리, 투명 배경, 우측 하단 배치

5. **하단 내비게이션 바**
   - 높이: 64px
   - 배경색: 흰색(#FFFFFF)
   - 그림자 효과: 위쪽으로 향하는 드롭 쉐도우
   - 아이콘:
     - 홈 (선택됨): #1E3A8A
     - 여행 계획: #9CA3AF
     - AI 컨설턴트: #9CA3AF
     - 예산 관리: #9CA3AF
     - 내 정보: #9CA3AF

## 상호작용
- **AI 컨설턴트 카드**: 탭하면 AI 컨설턴트 화면으로 이동
- **예산 관리 카드**: 탭하면 예산 관리 화면으로 이동
- **하단 내비게이션 바**: 각 아이콘 탭 시 해당 화면으로 이동
- **스크롤**: 콘텐츠가 화면보다 길 경우 세로 스크롤 지원
