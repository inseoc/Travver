# Travver - AI 여행 플래너 앱

## 프로젝트 개요

Travver는 AI 기반의 스마트 여행 계획 앱입니다. 사용자의 선호도와 예산에 맞춘 맞춤형 여행 일정을 생성하고, 실시간 AI 컨설팅을 제공합니다.

## 화면 설계

### 1. 스플래시 화면 (Splash Screen)
- **목적**: 앱 로딩 및 브랜드 노출
- **구성 요소**:
  - 앱 로고
  - 로딩 인디케이터
- **동작**: 2초 후 자동으로 온보딩/홈 화면으로 전환

---

### 2. 온보딩 화면 (Onboarding Screen)
- **목적**: 앱 사용법 안내 (최초 실행 시)
- **구성 요소**:
  - 3~4개의 슬라이드 페이지
  - 페이지 인디케이터
  - "시작하기" 버튼
- **내용**:
  1. AI 여행 플래너 소개
  2. 맞춤형 일정 생성 안내
  3. 예산 관리 기능 소개

---

### 3. 홈 화면 (Home Screen)
- **목적**: 메인 대시보드
- **구성 요소**:
  - 상단: 사용자 인사말
  - 중앙: 빠른 액션 버튼들
    - [새 여행 계획 만들기]
    - [AI 컨설턴트]
    - [내 여행 목록]
  - 하단: 최근 여행 계획 카드 목록

---

### 4. 새 여행 계획 입력 화면 (New Plan Input Screen)
- **목적**: 여행 정보 수집
- **입력 항목**:
  - 목적지 (검색/선택)
  - 여행 기간 (시작일 ~ 종료일)
  - 여행 인원
  - 예산 범위
  - 여행 스타일 (태그 선택)
    - 맛집 탐방 / 관광 명소 / 휴양 / 액티비티 / 쇼핑
- **버튼**: [AI 일정 생성하기]

---

### 5. 여행 계획 결과 화면 (Travel Plan Result Screen)
- **목적**: AI가 생성한 일정 표시
- **구성 요소**:
  - 일자별 탭 또는 타임라인 뷰
  - 각 일정 카드:
    - 시간
    - 장소명
    - 설명
    - 예상 비용
    - 지도 연동 (선택)
  - 하단 버튼:
    - [일정 저장하기]
    - [일정 수정 요청]

---

### 6. AI 컨설턴트 화면 (AI Consultant Screen)
- **목적**: 실시간 AI 채팅 상담
- **구성 요소**:
  - 채팅 메시지 목록
  - 메시지 입력창
  - 빠른 질문 버튼 (옵션):
    - "추천 맛집 알려줘"
    - "예산 줄이려면?"
    - "날씨 정보"
- **동작**: 실시간 스트리밍 응답 지원

---

### 7. 내 여행 목록 화면 (My Trips Screen)
- **목적**: 저장된 여행 계획 관리
- **구성 요소**:
  - 여행 계획 카드 목록
    - 목적지
    - 여행 기간
    - 상태 (예정/진행중/완료)
  - 카드 탭 시 상세 화면으로 이동
  - 스와이프 삭제 기능

---

### 8. 예산 관리 화면 (Budget Management Screen)
- **목적**: 여행 예산 추적
- **구성 요소**:
  - 총 예산 / 사용 금액 / 잔여 금액
  - 카테고리별 지출 차트 (원형 그래프)
    - 교통 / 숙박 / 식비 / 관광 / 기타
  - 지출 내역 추가 버튼
  - 지출 내역 목록

## 핵심 기능 목록

### MVP (Phase 1)
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 스플래시/온보딩 | 앱 초기 진입 화면 | P0 |
| 여행 계획 입력 | 목적지, 기간, 예산 등 입력 | P0 |
| AI 일정 생성 | GPT 기반 여행 일정 자동 생성 | P0 |
| 일정 저장/조회 | 생성된 일정 로컬 저장 | P0 |
| AI 채팅 상담 | 실시간 여행 관련 질의응답 | P1 |

### Phase 2
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 예산 관리 | 지출 추적 및 차트 시각화 | P2 |
| 지도 연동 | 장소 위치 지도에 표시 | P2 |
| 일정 공유 | 친구와 여행 계획 공유 | P2 |

### Phase 3
| 기능 | 설명 | 우선순위 |
|------|------|----------|
| 사용자 인증 | 로그인/회원가입 | P3 |
| 클라우드 동기화 | 여러 기기 간 데이터 동기화 | P3 |
| 푸시 알림 | 여행 일정 리마인더 | P3 |
